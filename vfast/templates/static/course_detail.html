目标受众

该课程将向你介绍深度学习。这项变革性的技术是许多人工智能领域创新发明背后的基础，它必将改变我们的未来生活。如果你想进入机器学习、人工智能领域，但缺乏相关技术背景，这门课程就是为你准备的！

先修要求

·学员需要有初级 Python 编程经验，并了解 Numpy。 如果你缺乏 Python 编程经验或相关数学基础，可以先学习机器学习(入门)课程，为继续学习深 度学习打造扎实的基础。 ·学员需要有一定的数学基础，掌握代数、偏导数和矩阵乘法。 ·在课程的一开始，我们会教你使用 scikit-learn 和 Numpy Python 库。在课程之后的部分， 我们会涉及到 Tensorflow，并用它来完成项目。

软/硬件要求

·MacBook / iMac / 任何 64 位操作系统、配有 8GB 以上内存的Windows PC ·Python 3+Jupyter Notebooks ·对于课程后半部分难度较大的项目，我们将为学生提供 AWS 实例(AWS instances)：申请 成功后，你将收到一封电子邮件，其中包含你可以在你的 AWS 账户中 使用的兑换码。
